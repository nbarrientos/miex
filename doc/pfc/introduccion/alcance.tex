\section{Alcance de este documento}

Este libro pretende explicar lo más breve y concisamente posible como se ha
desarrollado MIEX así como dar las claves necesarias para utilizarlo.

Además de una pequeña memoria que abarcará los aspectos no técnicos y el
historial del proceso de desarrollo, se incluye un capítulo completo destinado
a personal técnico (principalmente desarrolladores) y un manual para usuarios
finales.
