\section{Almacenamiento de la información}

A la hora de decidir el medio para almacenar los resultados obtenidos
rápidamente se descartó la opción de utilizar ficheros de texto plano, ya que
requerirían la construcción de analizadores adiccionales para, una vez se
quiera consultar los resultados, poder recuperar la información.

Finalmente se decidió utilizar una base de datos relacional para poder, en
cierta manera, diseñar una estructura lo más cómoda posible para dejar abierto
un gran abanico de posibilidades a la hora de consultar la información obtenida.

En el capítulo que se dedica a los aspectos técnicos, se explica la base de
datos utilizada así como la estructura de la misma.
