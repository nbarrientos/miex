\section{Primeros pasos}

Uno de los campos en los cuales se hace más hincapié dentro de la recuperación
de información es el análisis, tanto sintáctico como semántico, de textos para
posteriormente poder analizar los resultados obtenidos con diversos fines.

En general, los analizadores de lenguaje natural existentes tienen varias
desventajas. Una de ellas, es que no poseen interfaces que hagan fácil su manejo
y hacen perder mucho tiempo al usuario en escribir código adiccional o en leerse
mucha documentación para utilizarlos. Por otro lado, no suelen proporcionar
mecanismos para guardar la información extraída en medios flexibles para
posteriores consultas.

El objetivo de este proyecto es la elaboración de un envoltorio
(\textit{wrapper}) de uno de los analizadores más conocidos, el Stanford
Parser. Con éste envoltorio se pretende hacer que el manejo del mismo sea lo
más sencillo posible, para que, la persona que lo use, se pueda abstraer
totalmente del funcionamiento interno del software y poder analizar documentos
fácilmente. Para paliar la segunda desventaja de estos analizadores, se
proporcionará un sistema para el almacenamiento de la información en una base
de datos relacional.

Si lo que quiere es aprender directamente como utilizar MIEX, le aconsejo que
lea directamente al capítulo \ref{cap:manual}, donde se explica como ponerlo
en marcha. Si prefiere conocer el formato de las colecciones a analizar o la
información que se extrae de las mismas, siga leyendo este capítulo.
