\section{Objetivos}

Los objetivos de este proyecto son los siguientes:

\begin{itemize}
	\item Análisis de documentos XML.
	\item Comprobación de la integridad de los documentos XML.
	\item Utilización de ficheros de configuración.
	\item Uso del Stanford Parser para extraer toda la información posible
de los documentos analizados.
	\item Análisis de la información recabada por el Stanford Parser.
	\item Limpieza de información inútil.
	\item Diseño de una base de datos para guardar la información extraida.
\end{itemize}
