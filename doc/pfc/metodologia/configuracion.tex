\section{Los ficheros de configuración}

MIEX utiliza un fichero de configuración de tipo \textit{clave=valor} para
personalizar el comportamiento del programa. El fichero de configuración de
ejemplo que se distribuye con el software está bastante explicado, por lo que en
esta sección simplemente se hará referencia a él.

Las opciones de configuración existentes hasta la fecha son las siguientes:

\begin{Verbatim}[fontsize=\relsize{-2}]
# BDHostname
#   Type:     String
#   Default:  -
#   Required: Yes
#   Desc:     Self-explaining.
#   See also: BD*

# BDUser
#   Type:     String
#   Default:  -
#   Required: Yes
#   Desc:     Self-explaining.
#   See also: BD*

# BDPassword
#   Type:     String
#   Default:  -
#   Required: Yes
#   Desc:     Self-explaining.
#   See also: BD*

# BDName
#   Type:     String
#   Default:  -
#   Required: Yes
#   Desc:     Self-explaining.
#   See also: BD*

# Validate
#   Type:     Yes/No (boolean)
#   Default:  No
#   Required: No
#   Desc:     Tries to validate each collection file using XML Schema.
#   See also: XMLschemaURI

# XMLschemaURI
#   Type:     String
#   Default:  /usr/share/miex/schemas/default.xsd
#   Required: No
#   Desc:     Path to a XML Schema used to validate XML inputs.
#   See also: Validate

# Dump
#   Type:     Yes/No (boolean)
#   Default:  No
#   Required: No
#   Desc:     Creates one file per category containing sentences.
#   See also: DumpDir

# DumpDir
#   Type:     String
#   Default:  /tmp
#   Required: No
#   Desc:     Path to a directory where the files will be dumped.
#   See also: Dump

# CreateDB
#   Type:     Yes/No (boolean)
#   Default:  No
#   Required: No
#   Desc:     Injects the template sql file into the database before do any
insert. USE CAREFULLY.
#   See also: SQLSkeleton

# SQLSkeleton
#   Type:     String
#   Default:  /usr/share/miex/sql/skeleton.sql
#   Required: No
#   Desc:     Path to an SQL file containing the DB creation statements
#   See also: CreateDB

# Normalize
#   Type:     Yes/No (boolean)
#   Default:  No
#   Required: No
#   Desc:     If 'Yes' MIEX uses Porter and inserts normalized words as well.
#   See also: -
\end{Verbatim}

Una vez instalado el programa como se indica en el siguiente capítulo, se puede
tomar como plantilla el fichero de ejemplo que se encuentra en el directorio:

\texttt{\$PREFIX/share/doc/miex/examples/conf}
