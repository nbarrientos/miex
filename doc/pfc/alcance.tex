\chapter*{Alcance de este documento}

Este libro contiene la documentación del proyecto titulado \emph{\TituloPFC} y
pretende explicar lo más breve y concisamente posible como se ha desarrollado
así como dar las claves necesarias para utilizarlo.

Además de una pequeña memoria que abarcará los aspectos no técnicos y el
historial del proceso de desarrollo, se incluye un capítulo completo destinado
a personal técnico (principalmente desarrolladores) y un manual para usuarios
finales.

\section*{Palabras clave}

Recuperación de información, analizador sintáctico, analizador semántico,
Stanford Parser, SQL, Java.

\section*{Aclaración sobre el título}

Si bien el título del proyecto de cara a la distribución y utilización en el
marco universitario es es \textit{\TituloPFC}, su título original es
\textbf{\TituloPFCWeb}, por lo que, a lo largo de este documento, se hará
referencia al proyecto utilizando los dos títulos indistintamente.
