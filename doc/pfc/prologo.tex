\chapter*{Prólogo}

La importancia que posee la recuperación de información es vital en nuestros
días. A partir de un cierto volumen de texto, se hace imprescindible un
sistema organizativo que posibilite la localización de la información que se
precise en cualquier momento. Para facilitar la contrucción de los citados
sistemas organizativos se puede, y en muchos casos se debe, realizar un análisis
previo de los textos, tanto sintáctico como semántico.

En general, los analizadores de lenguaje natural existentes tienen varias
desventajas. Una de ellas, es que no poseen interfaces que hagan fácil su manejo
y por tanto hacen perder mucho tiempo al usuario en escribir código adicional o
en leerse mucha documentación para utilizarlos. Por otro lado, no suelen
proporcionar mecanismos para guardar la información extraída en medios flexibles
para posteriores consultas.

El objetivo de este proyecto es la elaboración de un envoltorio
(\textit{wrapper}) de uno de los analizadores más conocidos, el Stanford
Parser\footnote{http://www-nlp.stanford.edu/downloads/lex-parser.shtml}. Con
éste envoltorio se pretende hacer que el manejo del mismo sea lo más sencillo
posible, para que, la persona que lo use, se pueda abstraer totalmente del
funcionamiento interno del software y poder analizar documentos fácilmente. Para
paliar la segunda desventaja de estos analizadores, se proporcionará un sistema
para el almacenamiento de la información en una base de datos relacional.

Es importante que el usuario de este proyecto tenga en cuenta que las
capacidades de análisis tanto semántico como sintáctico están restringidas a las
propias limitaciones del analizador base que se toma, por lo que se aconseja
encarecidamente leer documentación sobre el Stanford Parser antes de continuar
con este libro.

Si lo que quiere es aprender directamente a utilizar MIEX, le aconsejo que
salte directamente al capítulo \ref{cap:manual}, donde se explica como ponerlo
en marcha. Si prefiere conocer el formato de las colecciones a analizar o la
información que se extrae de las mismas, comience por el capítulo
\ref{cap:introduccion}.
