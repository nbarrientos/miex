\section{Reconocimientos}

Permítame, por favor, ser más informal en esta esta sección.

Este proyecto ha sido íntegramente desarrollado utilizando software libre.
Gracias a todos los que habeis dedicado vuestro tiempo a desarrollar estas
maravillosas herramientas, en gran parte, este proyecto también es vuestro. No
os voy a poder nombrar a todos, pero si voy a nombrar algunos de los proyectos
en los que trabajais o habeis trabajado \begin{small};-)\end{small}.

Aquí van todos esos proyectos, sin ningún orden en particular:

\begin{quote}
\textit{GNU Make, Apache Ant, JSAP, MySQL Connector/J, SAX, Stanford Parser,
Vim, Umbrello, Texlive, KDE, Debian, Kile, Dia, Kernel.org, amaroK, Mutt,
Procmail, Fetchmail, y muchos más...}
\end{quote}

Mi manera de dar las gracias es liberando este proyecto para, modestamente,
hacer que la comunidad del software libre siga creciendo.

\begin{flushright}
\textit{El software, como el resto de la ciencia, libre.} -- René Mérou
\begin{small}<\url{h@es.gnu.org}>\end{small}
\end{flushright}
