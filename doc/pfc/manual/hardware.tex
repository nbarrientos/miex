\section{Requisitos hardware}

Como se advierte en la documentación del Stanford Parser el coste computacional
del análisis es muy alto, por lo que es recomendable disponer de una máquina de
prestaciones medias/altas para obtener resultados en un tiempo aceptable.

Por falta obvia de recursos, no se ha podido hacer pruebas en diferentes
configuraciones hardware por lo que, exceptuando el sistema operativo, tómense
los siguientes requisitos como una recomendación.

\begin{itemize}

\item \underline{Procesador}: Cualquier arquitectura es válida, mientras pueda
correr una distribución de GNU/Linux sobre ella. La frecuencia a la que funcione
el procesador y la cantidad de caché a convenir.

\item \underline{Memoria}: Entre 512 MiB y 2 GiB.

\item \underline{Disco}: A convenir (si va a instalar la base de datos en el
equipo local, evalue el número de colecciones a analizar para elegir una
capacidad adecuada).

\item \underline{Sistema Operativo}: Se recomienda cualquier distribución de
\textbf{GNU/Linux}, aunque cualquier sistema operativo bajo el se pueda correr
una máquina virtual de Java podría servir.

\item \underline{Otros}: A convenir.

\end{itemize}
