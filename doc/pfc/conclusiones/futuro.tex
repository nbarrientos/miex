\section{El futuro}

Como se ha comentado anteriormente, en la medida de lo posible, este proyecto se
continuará manteniendo. Al ser un proyecto libre, el autor espera que pueda
ser utilizando por mucha más gente para este uso en concreto o para tomarlo como
base y modificarlo.

Si algún lector de este libro está interesado en seguir mejorando MIEX aquí se
recogen una serie de ampliaciones que pueden resultar interesantes.

\begin{itemize}
\item Desarrollo de una aplicación web
utilizando AJAX\footnote{http://es.wikipedia.org/wiki/AJAX} para la consulta
dinámica de la base de datos, donde el investigador pueda personalizar de una
manera sencilla las consultas.

\item Añadir un sistema para que en tiempo de ejecución se pueda personalizar
el nombre de las etiquetas de los ficheros XML a tratar y así evitar que
esté \textit{hardcoded} en el código fuente.
\end{itemize}
