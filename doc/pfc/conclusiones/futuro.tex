\section{El futuro del proyecto}

Como se ha comentado anteriormente, en la medida de lo posible, este proyecto se
continuará manteniendo. Al ser un proyecto libre, el autor espera que pueda
ser utilizado por mucha más gente para este uso en concreto o para tomarlo como
base y modificarlo.

Si algún lector de este libro está interesado en seguir mejorando MIEX aquí se
recogen una serie de ampliaciones que pueden resultar interesantes.

\begin{itemize}

\item Desarrollo de una aplicación web
utilizando AJAX\footnote{http://es.wikipedia.org/wiki/AJAX} y algún lenguaje
de uso frecuente en web (por
ejemplo PHP\footnote{http://es.wikipedia.org/wiki/PHP}) para la consulta
dinámica de la base de datos, donde el investigador pueda construir de una
manera sencilla las consultas sin tener prácticamente conocimientos de SQL.

\item Escribir un parche para que, en tiempo de ejecución, se pueda personalizar
el nombre de las etiquetas de los ficheros XML a tratar (por ejemplo,
utilizando el fichero de configuración) y así evitar tener que tocar el código
fuente para modificarlas.

\item Internacionalización del software utilizando una biblioteca con el mismo
propósito que GNU Gettext\footnote{http://www.gnu.org/software/gettext/}, como
por ejemplo
Java Intl\footnote{http://java.sun.com/javase/technologies/core/basic/intl/}.

\item Sacando ventaja del punto anterior, localizar el programa a distintos
idiomas, entre ellos, el castellano.

\item Mejorar MIEX para extraer propiedades lingüísticas de palabras que sirvan
de atributos para el agrupamiento de palabras.

\end{itemize}
